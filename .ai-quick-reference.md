### Database
- **Tool**: **Flyway** (ADR-013), driven via the Makefile
- **NOT**: Liquibase, custom runners, the archived Rust CLI, or the superseded Python `db-migrate`
- **Run**: `make migrate` (or `make db-reset`)

## 🚫 Never Do

1. ❌ Suggest Liquibase or a custom migration runner (this project uses **Flyway**, ADR-013)
2. ❌ Invoke the archived Rust CLI or the superseded Python `db-migrate`
3. ❌ Use `any` in TypeScript
4. ❌ Skip `DatabaseFactory.dbQuery`
5. ❌ Return Exposed entities
6. ❌ Add deps without version catalog

## ✅ Always Do

1. ✅ Use `DatabaseFactory.dbQuery { }`
2. ✅ Log to `audit_log`
3. ✅ Return DTOs
4. ✅ Reference `gradle/libs.versions.toml`
5. ✅ Use TypeScript strict mode
6. ✅ Use Tailwind utilities
